Dagger

Straightforward site generation for C#/.NET

Quickstart

Installation, creating and serving an example site.

1. Install the .NET SDK

2. Install Dagger

                
                    > dotnet tool install --global Jokocide.Dagger

3. Create a Dagger project

                
                    > dagger new mysite
                    
                    Dagger new mysite
                    ------
                    Created Dagger project at C:\Users\jkoen\mysite

4. Serve

                
                    > dagger serve mysite
                    
                    Dagger serve mysite
                    ------
                    Serving project on http://localhost:8000
